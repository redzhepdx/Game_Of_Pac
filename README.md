```SOIL```
```GL GLU glut```

### Compile and Run
- Do it only once!

``` mkdir -p out/build ```

``` cd out/build/ ```

``` cmake -S ../../ -B .```

- After every update!

``` make ```

- Start the game

``` ./GameOfPack```

### Controls

```W : Up```
```D : Down```
```A : Left```
```S : Right```

```Q : Anti Clock Wise Rotate```
```E : Clock Wise Rotate```

```X : Shoot```

```Esc : Quit```
